# Amplify AppSync Chat

Full stack chat app built with GraphQL and React

## To deploy this app

1. Clone the repo

2. Change into the `chat-client` directory.

3. Install the dependencies:

```sh
npm install

# or

yarn
```

4. Initialize and deploy the Amplify back end:

```sh
amplify init

amplify push
```

5. Run the app

```sh
npm start
```
